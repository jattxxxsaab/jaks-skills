# IHSS Timesheet Entry Procedure

## Overview
Fill and submit IHSS timesheets for **Parminder Bhullar** (provider) → **Sanjit Bhullar** (recipient) on the Electronic Services Portal.

## Website
- URL: https://etimesheets.ihss.ca.gov/login
- Username: `YOUR_PROVIDER_USERNAME`
- Password: `YOUR_PROVIDER_PASSWORD`

## Schedule
- **1st pay period:** Feb 1-15 → enter & submit on the **15th**
- **2nd pay period:** Feb 16-end of month → enter & submit on the **last day of the month**
- Timesheets can only be **submitted on the last day of the pay period**
- Before that, you can save but not submit

## Steps

### 1. Log In
- Go to https://etimesheets.ihss.ca.gov/login
- Enter username and password
- Click Login
- Dismiss the security info popup (click OK)

### 2. Navigate to Timesheet Entry
- Click **TIMESHEET ENTRY** on the dashboard (or select from Time Entry menu)
- Select recipient: **SANJIT BHULLAR**

### 3. Select Pay Period
- Use the "Pay Period - Payment Type" dropdown to select the current pay period

### 4. Read Available Hours
- Note the **Available Hours for [Month]** (e.g., "112(H) 16(M)")
- These are **monthly** hours

### 5. Calculate Daily Hours
- Convert available hours to total minutes: `(hours × 60) + minutes`
- Divide by **number of days in the month** (not pay period)
- Result = hours and minutes per day
- Example: 112H 16M = 6,736 min ÷ 28 days = 240 min = **4H 00M/day**

### 6. Enter Time for Each Day
- Expand each Workweek (1, 2, 3 as needed)
- For each day in the pay period, enter the calculated hours and minutes
- Example: 4 in Hours, 00 in Minutes for each day
- **Save** each workweek after filling it

### 7. Submit (Only on Last Day of Pay Period)
- If today is the **15th** or **last day of the month**: click **Submit Timesheet**
- If before that date: only **Save** — do NOT submit

## Notes
- Available hours are per month, shared across both pay periods
- Recipient ID: YOUR_RECIPIENT_ID
- Timesheet Type: IHSS
- If hours don't divide evenly, distribute remainder across days (e.g., add 1 extra minute to some days)
